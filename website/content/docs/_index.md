tenantplane is an open source virtual Kubernetes tenant platform for teams that
need multi-tenancy that is **secure by design, transparent, and easy to
operate**. These docs take you from a first tenant on a kind cluster through the
concepts and reference material behind them.

New here? Start with the [Introduction](/docs/introduction/), then run the
[Quickstart](/docs/quickstart/).
